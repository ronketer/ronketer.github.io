MapReduce Framework

Built a multi-threaded MapReduce engine from scratch in C++17. Threads claim work via a shared std::atomic<uint64_t> counter (lock-free work-stealing), eliminating a thread-pool queue. Per-thread intermediate vectors prevent emit contention; a two-phase pthread barrier synchronizes map → shuffle → reduce. Live job-state reporting via atomic counters, no stalling worker threads.

C++17 pthreads std::atomic pthread_barrier

Stereo Mosaicing

Image processing pipeline that turns a panning video into a wide-angle panorama. Tracks Shi-Tomasi corners across frames with Lucas-Kanade optical flow to estimate per-frame translation, vertically stabilizes the sequence, then composites a panorama by sweeping a virtual slit through a space-time volume — different slit positions produce different perspective outputs from the same input.
