## Secure REST API & DevOps CI/CD Pipeline

**Technologies:** Node.js, Git, GitHub Actions, Express, Jest

This project demonstrates modern DevOps practices combined with secure API development.

## Project Overview

I built a production-ready REST API with comprehensive security features and automated deployment pipelines. The project showcases best practices in both application security and DevOps automation.

## Key Features

### Automated CI/CD Pipeline

- Automated DevOps CI/CD pipelines via GitHub Actions
- Matrix testing across multiple Node.js versions and environments
- Reduced deployment risk through comprehensive pre-deployment checks

### Comprehensive Testing

- Achieved >80% global test coverage
- Developed 5 dedicated test suites using Jest and Supertest
- Implemented tests for edge cases and regression prevention

### Security Architecture

- JWT-based authentication system
- HTTP security headers (CORS, CSP, etc.)
- Input validation and schema enforcement
- Protection against common web vulnerabilities

## Technical Implementation

### API Architecture

```
├── Authentication (JWT)
├── Input Validation (Schema-based)
├── Rate Limiting
├── Error Handling
└── Security Middleware
```

### CI/CD Pipeline Stages

1. **Code Quality:** Linting and formatting checks
2. **Testing:** Unit and integration tests
3. **Security:** Dependency scanning
4. **Deployment:** Automated release to production

## DevOps Highlights

- **Artifact Management:** Proper build artifact handling
- **Environment Configuration:** Secure secrets management
