## Autonomous Multi-Agent LLM Pipeline

**Technologies:** Python, AutoGen, Ollama, Pydantic

This project showcases my work in building an autonomous multi-agent system powered by Large Language Models (LLMs).

## Project Overview

I developed a local-first multi-agent AI system using Microsoft AutoGen and Python to automate the extraction and synthesis of unstructured notes into Anki flashcards. This project demonstrates practical applications of agentic AI design patterns while prioritizing data privacy through local LLM inference.

## Key Features

### Agentic Design Patterns

- Implemented Tool Use, Reflection, and Multi-Agent Collaboration patterns
- Utilized local LLMs via Ollama for privacy-preserving inference
- Ensured strict output formatting using Pydantic validation

### Human-in-the-Loop Workflow

- Architected an approval workflow that keeps humans in control
- Integrated external APIs for seamless data flow
- Built an automated, end-to-end knowledge processing Proof of Concept

## Technical Highlights

- **Multi-Agent Orchestration:** Coordinated multiple AI agents working together to process, analyze, and transform unstructured data
- **Local-First Architecture:** All LLM inference runs locally via Ollama, ensuring complete data privacy
- **Type-Safe Outputs:** Leveraged Pydantic for strict schema validation and reliable output formatting
- **API Integration:** Connected external services to enable end-to-end automation

## Skills Demonstrated

- Large Language Model integration and prompt engineering
- Multi-agent system architecture
- Python development with modern tooling
- Data privacy and local AI deployment
